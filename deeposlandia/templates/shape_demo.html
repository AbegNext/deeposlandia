Description

In this dataset, there are three labels, and consequently the model aims at answering one question for each:

  • Is this a square on the picture?
  • Is this a circle on the picture?
  • Is this a triangle on the picture?

Remark about model training

The model used here has been trained during 10 epochs with a set of 18000 randomly generated images. It has been validated with a smaller set of 2000 randomly generated images (the validation occurred at the end of each training epoch), in order to ensure that learned features are as generic as possible.
